##------------------Casita 3D---------------##

This project exemplifies basic implementation and usage of OpenGL 2.0 Library for an Android App

Includes operations like:

 - Moving Objects
 - Rotating Objects
 - Scaling Objects
 - Sheering Objects
 - Moving Animations

Also has features like:

 - Zoom in/out with modal "pinch" (2 fingers)
 - Camera reprojection based on screen interaction
 - Wireframe Mode
